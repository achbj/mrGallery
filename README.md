# Mr Gallery 🖼️

**Mr Gallery** is an intelligent, privacy-first photo and video gallery application. It uses advanced AI (Face Recognition, Object Detection) to scan your photos and videos completely **locally** on your own computer. None of your media is ever uploaded to the cloud!

---

## 🚀 How to Download and Run (No technical skills required!)

### For Windows Users
1. **Download the App:** Go to the [Releases page](../../releases) (or the [Actions tab](../../actions) if a release isn't published yet) and download `MrGallery-Windows.zip`.
2. **Extract:** Right-click the downloaded `.zip` file and select "Extract All...".
3. **Run:** Open the extracted folder and double-click **`mr-gallery-win_x64.exe`**.
   *Note: A black console window might appear briefly in the background—this is the AI engine starting up. Just ignore it!*

### For Mac Users
*(macOS app builder coming soon)*
If you are on a Mac, you can run the app using the command line for now:
1. Open Terminal and navigate to the folder where you downloaded this code.
2. Run `npm install` to install dependencies.
3. Run `npm run app:dev` to launch the app!

### For Linux Users
*(Linux app builder coming soon)*
Linux users can use the same terminal commands as Mac users:
1. `npm install`
2. `npm run app:dev`

## 🌟 Features
* **Lightning Fast Timeline:** View thousands of your photos grouped beautifully by date.
* **AI Face Recognition:** The app automatically finds faces in your photos and groups them by person.
* **Find Duplicates:** Instantly scan for duplicate photos or videos to free up hard drive space!
* **Works Completely Offline:** Because it runs entirely on your local machine, it is 100% private and works without an internet connection.

## 📁 Managing Folders
1. Click the **"Manage Folders"** icon in the sidebar.
2. Click **"Add folder"** to choose a folder on your computer that contains photos or videos.
3. The app will immediately start scanning the folder and importing everything into your timeline.

Enjoy your completely private, intelligent gallery!
